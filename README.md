## Command and Description

| Command | Description |
| ----------------------- | -------------------------- |
| `/sys`                  | Show system information    |
| `/ip`                   | Get public IP address      |
| `/run <command>`        | Execute system command     |
| `/screen`               | Take a screenshot          |
| `/record <seconds>`     | Record screen              |
| `/search <query>`       | Search and capture screen  |
| `/get <file_path>`      | Download file              |
| `/photo <photo_path>`   | Download image             |
| `/folder <folder_path>` | Download folder as zip     |
| `/encrypt <path>`       | Encrypt file or folder     |
| `/decrypt <path> <key>` | Decrypt file or folder     |
| `/lock`                 | Disable keyboard and mouse |
| `/unlock`               | Enable keyboard and mouse  |
| `/close`                | Stop client                |
